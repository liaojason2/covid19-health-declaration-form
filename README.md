# 快速實聯制
因應政府在疫情實行的實聯制政策，進出各大公共場所時需要填寫實聯制表單，確保確診後可以透過實聯制的資料找到接觸者。本專案透過綁定社群帳號與實聯制所需資料，解決每次進入店家時都要花時間在填寫資料上的問題。
